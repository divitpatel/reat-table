# Grid POC
 
Grid is feature rich datagrid designed for the major JavaScript Frameworks.

## How to build this project (for development)
1. `clone` the repo
2. `cd` into the repo
3. `npm install`
4. `npm start` 
5. Navigate to `localhost:3000` in your browser
